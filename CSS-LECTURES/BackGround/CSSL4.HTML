CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML,
including aspects like layout, colors, and fonts.

JAVA

Java is a versatile, object-oriented programming language known for its platform independence and wide use in developing applications, from mobile to enterprise systems.

HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages, defining the structure and elements within a document.
